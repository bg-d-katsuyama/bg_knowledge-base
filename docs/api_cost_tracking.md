## 計測方法

- Anthropicのコンソール（Console > Usage）から月次レポートを取得
- 補足としてCloud Loggingの `jsonPayload.event="claude_api_call"` を集計
- 各リクエストには以下のメタデータを構造化ログで記録：
  - `model`: 使用モデル
  - `input_tokens`: 入力トークン数
  - `output_tokens`: 出力トークン数
  - `cache_read_tokens`: キャッシュ読込トークン
  - `pipeline`: 呼び出し元パイプライン名
